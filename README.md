## 🚀 Featured Projects

### 🔭 [ExoScope](https://github.com/saanvik/exoscope)
> **Exoplanet Detection via Machine Learning**

An end-to-end ML pipeline for detecting exoplanet candidates from stellar light curve data. Engineered features from transit signatures and trained classification models to separate true planetary transits from instrumental noise and astrophysical false positives.

---

### 🦟 [MosquitoHSM](https://github.com/saanvik/mosquitoHSM)
> **Habitat Suitability Modeling of *Aedes albopictus***

A species distribution model using Maximum Entropy (MaxEnt) to predict the spread and invasion risk of *Aedes albopictus* — the tiger mosquito — under current and projected climate scenarios. Pulls together ecological occurrence records, bioclimatic variables, and geospatial layers to decide where disease prevention measures and green stormwater infrastructure is needed most.
